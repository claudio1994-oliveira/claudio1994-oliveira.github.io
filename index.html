Minha EBD - Portal do aluno

O Minha EBD foi criado para registrar a presença de alunos na Escola Bíblica Dominical e nas atividades escolares. O registro das presenças geram pontos em um sistema de gameficação para que os gestores possam mensurar a adesão (presenças) e bonificar aqueles com mais frequências.

O sistema foi desenvolvido com PHP, utilizando o framework Laravel e o Livewire para a construção de componentes dinâmicos. O banco de dados utilizado é o MySQL e o front-end foi construído com Bootstrap.

Componente Router

O Componente Router é uma ferramenta simples e flexível para gerenciar rotas em aplicativos PHP. Este componente permite que você defina rotas e associe funções ou métodos de controladores a elas, facilitando a criação de aplicativos da web com URLs amigáveis.

O pacote foi desenvolvido em PHP e seus testes em PHP Unit. Um dos recursos são: fácil definição de rotas, suporte para parâmetros nas rotas e funções ou métodos de controladores associados a rotas

My PHP MVC

O projeto é desenvolvido em PHP e seus testes em PHP Unit. Considere em contribuir com o projeto e se divirta programando em PHP.
